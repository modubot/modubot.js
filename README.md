Modubot
=======

Modubot, the scalable, extensible IRC Bot.

Installation
------------

### Using modubot-js.cli

    npm install -g modubot.js-cli
    modubot new bot-name

### Manually

    npm install -g grunt-cli
    npm install
    grunt
    npm start

Documentation
-------------

You can view Modubot's documentation [here](https://modubot.net/docs.html).
